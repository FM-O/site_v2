Compétences en développement web

Compétences développement front-end

Javascript et frameworks associés

Mes compétences en Javascript s'améliorent de plus en plus étant donné qu'il s'agit aujourd'hui de mon langage de prédilection. J'ai également de bonnes connaissances sur la librairie Jquery. Les implémentations liées à mon site concernent principalement des ajouts d'animations, de filtres (calques au-dessus d'une page ou d'une partie de page), ainsi que l'envoie de messages en Ajax (JS Natif).
Je compte aussi évoluer vers le Framework Angular JS une fois que la version 2 sera disponible, celle-ci s'éloignant de la version 1.x dans son fonctionnement.

CSS, pré-processeurs et frameworks associés

Je possède à ce jour de bonnes compétences en CSS3 et sur le pré-processeur SASS et notamment le framework Compass. Celui-ci facilite grandement l'écriture du CSS et ajoute de nombreuses mixins et fonctions utiles à SASS. Actuellement, sur mon site, ce sont plus de 4000 lignes de CSS générées à l'aide de Compass...

HTML, et templating

Expérience sur le moteur de template PHP : Twig. Connaissances des bonnes pratiques W3C ainsi que des meta Open Graph afin d'affiner le référencement naturel. Connaissance de l'intégration Java J2EE (JSP, JSF).
Expérience sur les Canvas et les nouveautés HTML5 (balises videos, audios...).

Micro-blog réalisé en PHP procédural

Java J2EE, JSP, JSF

Je possède quelques bases en Java, travaillant essentiellement avec ce langage au sein de l'entreprise SQLI. Mes compétences sont concentrées essentiellement autour de l'utilisation des JSP. Cependant, j'ai de bonnes notions, sur l'articulation d'un projet d'application web créé avec J2EE.

Compétences sur les CMS

Prestashop

Maîtrise correcte de Prestashop, j'ai pu développer une partie d'un site sous ce CMS à cette adresse : voir le site. Je possède aussi quelques compétences sur le moteur de template Smarty de Prestashop.

WordPress

Wordpress est le premier CMS vers lequel je me suis tourné. Pratique, facile d'utilisation, il permet aussi de nombreuses personnalisations faciles à mettre en place. J'ai pu notamment créé des templates de page pour faciliter l'édition d'un blog lié au précédent site (le blog n'existe plus à ce jour). Développer un site sous ce CMS ne me pose donc aucun problème.

Jahia

Jahia est un CMS récent très bien pensé. Il est orienté Java et s'utilise donc pour de nombreux cas d'application. Je l'utilise actuellement au cours de mon stage au sein de SQLI Bordeaux. Le développement sous ce CMS est plus difficile que sous Prestashop ou Wordpress. Dans le cas du projet que je poursuis en entreprise, il devra servir de portail applicatif permettant d'aggréger plusieurs applications clientes.
